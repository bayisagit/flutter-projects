# Facebook Home Page Clone

This is a simple Flutter application that demonstrates a Splash Screen and a basic Facebook Home Page clone.

## Getting Started

1.  **Prerequisites**: Ensure you have Flutter installed and set up.
2.  **Dependencies**: Run `flutter pub get` to install the dependencies.
3.  **Run**: Run the app using `flutter run`.

## Project Structure

-   `lib/main.dart`: Contains the main application code, including the Splash Screen and Home Page.
-   `assets/images/facebook_logo.png`: The Facebook logo used in the app.
-   `pubspec.yaml`: Configuration file for dependencies and assets.

## Features

-   **Splash Screen**: Displays the Facebook logo for 3 seconds.
-   **Home Page**: A simple list view mimicking the Facebook feed with a blue AppBar and sample posts.
